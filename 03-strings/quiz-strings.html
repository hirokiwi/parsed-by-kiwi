12
C++代码 string s = "GESP考试";，s 占据的字节数是（ ）
A10
B8
C8或10
D取决于计算机采用什么编码
✓ 答案 D

解析

中文字符在不同编码下占不同字节数：
ASCII（英文/符号）：每个字符占 1 字节 GBK 编码：每个中文字占 2 字节 UTF-8 编码："GESP"=4字节 + "考试"=6字节（每汉字3字节）= 10字节
字节数取决于编码方式。

string str = "Hello"; str.replace(5, 5, "C++"); cout << str;
string str = "Hello"; str.replace(1, 5, "C++"); cout << str;
string str = "HelloC++World"; str.replace(5, 5, ""); cout << str;
A第1个程序
B第2个程序
C第3个程序
D第4个程序

16
C++中 string 的 == 运算符比较的是字符串的内存地址，而非内容。
✗ 正确
✓ 错误
✓ 答案 ✗ 错误

解析

这道题的说法是错误的。
string 的 == 运算符比较的是字符串的内容，不是内存地址 即使两个 string 对象地址不同，只要内容相同，== 就返回 true
区分：
string a = "abc", b = "abc";
a == b → true（比较内容）
&a == &b → false（比较地址）

19
C++中，string 类型的字符串末尾一定包含 '\0' 字符。
✗ 正确
✓ 错误
✓ 答案 ✗ 错误

解析

这个说法是错误的。
C++ string 是高级字符串类，内部自己管理内存，不依赖 '\0' 作为结束标记 string 用 length() / size() 记录长度，不需要用 '\0' 判断结尾
区分：
C++ string：不需要 '\0' 结尾，长度由对象属性决定
C 风格字符串：必须以 '\0' 结尾，否则无法判断长度
20
string 对象的 size() 和 length() 方法返回的值总是相同的。
✓ 正确
✗ 错误
✓ 答案 ✓ 正确

解析

这个说法是正确的。
size() 和 length() 在 C++ string 中是完全等价的 length() 是 string 类为了符合人类习惯而额外提供的 size() 是从 STL 容器继承来的接口，两者底层调用的是同一个函数
所以 s.size() == s.length() 恒成立。
